# VISIT INDIA — Premium Landing Page

A refined, atmospheric, deeply cinematic landing page for a premium 10-day curated travel experience across Delhi, Jaipur, and Kerala.

## Tech Stack
- Next.js 14 (App Router, JSX)
- Tailwind CSS
- Framer Motion
- Lenis (smooth scroll)
- Next/Image

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Image Assets
All images are sourced from Unsplash. Replace the URLs in the components with your own production imagery:
- `components/Hero.jsx` — Hero background & saree figure
- `components/Hero.jsx` — Polaroid card images (5 cards)
- `components/About.jsx` — Timeline destination photos (6 images)
- `components/Contact.jsx` — Contact section background

## Customization
- Colors: Edit `tailwind.config.js`
- Typography: Edit `app/layout.jsx` (Google Fonts) and `tailwind.config.js`
- Motion: Edit individual component Framer Motion settings
